# Finsight – Future Enhancements

This document outlines features, improvements, and ideas planned for future versions of the application.

## 1. Features to Add Later

- Recurring Transactions – Ability to mark transactions as recurring (e.g., monthly rent, subscriptions).
- Budgeting & Alerts – Set monthly budgets per category and receive notifications when close to limits.
- Multi-currency Support – Automatic conversion and reporting in preferred currency.
- Advanced Dashboard – Visual charts and trends over time (income vs expense, category-wise).
- Export & Import – Export transactions to CSV/PDF and import from external sources.

## 2. Ideas for Version 2

- User Roles & Admin Panel – Admin users to manage system-wide settings, categories, and users.
- Collaborative Accounts – Shared accounts for families or teams with controlled access.
- Mobile App / PWA – Cross-platform mobile app with offline capabilities.
- AI-based Insights – Intelligent suggestions on spending, savings, and investment opportunities.

## 3. Improvements

- Enhanced Security – Full Spring Security integration, password policies, and token invalidation on logout.
- Performance Optimization – Indexing, caching, and query optimization for large datasets.
- Error Handling & Logging – Centralized error handling, improved logging, and user-friendly error messages.
- Unit & Integration Tests – More comprehensive test coverage for all modules.
- Code Quality & Documentation – Maintain high code standards, API documentation, and developer guides.
