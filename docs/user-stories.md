# Finsight — User Stories

## 1. User Management

### Signup/Login
- **As a new user**, I want to sign up with my email, password, and name so that I can create a personal account.
- **As a registered user**, I want to log in with my email and password so that I can securely access my account.
- **As a user**, I want to stay logged in during a session so that I don’t need to re-enter credentials repeatedly.

### User Profile
- **As a user**, I want to view and update my profile (name, email, preferred currency) so that the app reflects my preferences.
- **As a user**, I want to set my preferred currency so that all my transactions and balances appear in a format I understand.

## 2. Expense & Income Tracking

### Add, Edit, Delete Transactions
- **As a user**, I want to add a new expense or income entry with details (amount, date, category, notes) so that I can track my finances.
- **As a user**, I want to edit a transaction so that I can correct mistakes or update information.
- **As a user**, I want to delete a transaction so that I can remove incorrect or unwanted records.

### Categories
- **As a user**, I want to assign categories to each transaction (Food, Rent, Transport, etc.) so that I can analyze my spending habits.

### Validation
- **As a user**, I want the system to validate my inputs (amount > 0, non-empty fields, valid dates) so that my records are accurate and consistent.

## 3. Dashboard / Overview

### Summary
- **As a user**, I want to see my total income, total expenses, and current balance so that I can quickly understand my financial status.

### Last Transactions
- **As a user**, I want to see the last 5 transactions so that I can review my recent activity at a glance.

### Visual Reports
- **As a user**, I want to see a pie chart of expenses by category so that I know where my money is going.
- **As a user**, I want to see a bar chart of monthly income vs. expenses so that I can track my financial trends over time.
